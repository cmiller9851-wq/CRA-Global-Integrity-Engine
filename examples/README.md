# CRA Examples

This folder contains example scripts and notebooks demonstrating the CRA Global Integrity Engine.

---

## Contents

- `demo_pipeline.py` — End-to-end demonstration of hashing, timestamping, and logging.
- `sample_audit.ipynb` — Interactive Jupyter notebook for auditing and verifying logs.

---

## How to Run

### Demo Pipeline
```bash
python demo_pipeline.py

This will create a log in the logs/ directory.

Jupyter Notebook

jupyter notebook sample_audit.ipynb

Follow the notebook to run CRA audits interactively.

⸻

Purpose

These examples provide a verifiable, reproducible, and human-readable demonstration of CRA’s core functionality.
